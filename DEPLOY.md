# 🚀 AttendX – Deploy Guide (Super Easy!)

---

## 📦 What's in this folder?

```
attendance-platform/
  ├── index.html   ← The main file (open this!)
  ├── style.css    ← Makes it look beautiful
  ├── app.js       ← Makes it work
  └── DEPLOY.md    ← This guide
```

---

## ✅ METHOD 1 – Open on YOUR Computer (Zero steps!)

1. Open the folder on your computer
2. Double-click **index.html**
3. It opens in your browser — DONE! 🎉

> ⚠️ Camera/location may ask for permission — click Allow!

---

## 🌐 METHOD 2 – Put it ONLINE FREE (Netlify – Takes 2 minutes)

### Step 1 — Go to Netlify
Open your browser and go to:
👉 **https://netlify.com**

### Step 2 — Sign Up (Free)
- Click **Sign Up**
- Use your email (or GitHub account)

### Step 3 — Deploy!
- After login, you'll see a big box that says **"Drag and drop your site folder here"**
- Just drag the **whole folder** (attendance-platform) into that box!
- Wait 10 seconds... and you get a link like: `https://sparkly-panda-123.netlify.app`
- **Share that link with anyone!**

That's it! 🎉

---

## 🐙 METHOD 3 – GitHub Pages (Also Free, takes 5 minutes)

### Step 1 — Make a GitHub account
👉 **https://github.com** → Sign Up (free)

### Step 2 — Create a new repository
- Click the **+** button (top right) → **New repository**
- Name it: `attendx`
- Click **Create repository**

### Step 3 — Upload files
- Click **Upload files**
- Drag all 3 files (index.html, style.css, app.js) into the box
- Click **Commit changes** (green button)

### Step 4 — Turn on GitHub Pages
- Go to **Settings** tab of your repo
- Scroll down to **Pages**
- Under "Source" → select **main** branch → click **Save**
- Wait 1 minute → you get a link like: `https://yourname.github.io/attendx`

## 🆘 Problems?

- **Camera not working?** → Click "Allow" when browser asks for permission
- **Location not working?** → Click Allow OR it auto-uses Hyderabad as demo
- **File won't open?** → Make sure all 3 files (index.html, style.css, app.js) are in the SAME folder
- **Netlify error?** → Make sure index.html is in the ROOT of the folder (not inside another folder)
